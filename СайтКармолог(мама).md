## 3. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ TILDA

> ⚠️ Это не обычный HTML-файл. Он импортируется в Tilda — соблюдай все правила ниже.

**Структура файла:**
- ❌ Не добавлять `<!DOCTYPE>`, `<html>`, `<head>`, `<body>` — Tilda управляет ими сама
- ✅ Файл начинается с `<style>`, затем секции, в конце `<script>`
- ✅ Шрифты подключать через `@import` внутри `<style>`, не через `<link>`
- ✅ Каждая секция — `<section id="...">` с явным `padding` и `background`
- ✅ Максимальная ширина контента — `1200px`, центрирование `margin: 0 auto`

**Запрещено:**
- ❌ `position: fixed` — навигацию строит Tilda, навбар не нужен
- ❌ Внешние JS-библиотеки — только нативный JS
- ❌ `box-shadow` тяжёлые, градиенты

**Скругления (`border-radius`):**
- Единый радиус на весь сайт: все прямоугольные блоки — фото/медиа, кнопки, карточки услуг и отзывов — `12px`
- Поля формы и разделители — без скругления
- Круглые элементы (аватары, кнопки карусели, Play) — `50%`

**Адаптивность:**
- Breakpoint: `768px`
- На мобайле: H1 → `clamp(28px, 7vw, 38px)`, H2 → `32px`, все колонки → одна, карусели листаются свайпом (кнопки ← → скрыты)

**Имя файла:** `tilda-karmolog.html`

**Правила работы с фото:**
- Все изображения — `object-fit: cover`, никогда `contain`

- Портретные фото (Hero, About, Contact) — показывать от пояса и выше, не обрезать лицо
- На мобайле Hero-фото уходит под текст (стек), высота `60vw`

## 5. ЦВЕТОВАЯ ПАЛИТРА

```css
:root {
  /* Фоны */
  --bg-primary:   #FDF9F4;  /* основной фон страницы */
  --bg-section:   #F5EFE6;  /* фон чередующихся секций */
  --bg-card:      #E8D9C5;  /* фон секции услуг, шапки, футера, аватаров */
  /* поверхность карточек услуг — на тон светлее bg-card */
  --surface-card: #EFE2CF;  /* фон карточек услуг (лежат на секции #E8D9C5) */

  /* Акценты */
  --accent-main:  #C4845A;  /* CTA-кнопки, hover, декор */
  --accent-mid:   #8B6E5C;  /* заголовки H2–H3, иконки */
  --accent-dark:  #3D2B1F;  /* акцентный тёмный: H1, H3, ввод в форме, аватар */

  /* Вспомогательные */
  --text-muted:   #7A6255;  /* основной текст: абзацы, описания */
  --tag-text:     #7A5C3A;  /* метки на светлом фоне */
  --text-on-cta:  #FFFFFF;  /* текст на кнопках */
  --divider:      #E8D9C5;  /* разделители, бордеры */
}
```

**Правила сочетаний:**

| Элемент | Фон | Текст |
|---------|-----|-------|
| Кнопка CTA (главная) | `#C4845A` | `#FFFFFF` |
| Кнопка вторичная | прозрачный | `#C4845A` + обводка `#C4845A` |
| Абзацы / описания (основной массив текста) | `#FDF9F4` / `#F5EFE6` | `#7A6255` (`--text-muted`) |
| Акцентный тёмный текст (H1, H3, ввод в форме, инициалы аватара) | — | `#3D2B1F` (`--accent-dark`) |
| H2 | — | `#8B6E5C` (`--accent-mid`) |
| Цитата | — | `#8B6E5C` (`--accent-mid`) |
| Карточка услуги | `#EFE2CF` | абзац `#7A6255`, цена `#C4845A` |
| Теги / бейджи / метки | `#E8D9C5` | `#7A5C3A` |
| Разделители | `#E8D9C5` | — |

> `#3D2B1F` (`--accent-dark`) — это акцентный тёмный цвет для заголовков и выделенных строк, **не** цвет всех абзацев. Основной массив текста (абзацы, описания) идёт цветом `#7A6255` (`--text-muted`).

**Запрещено:**
- ❌ Чёрный `#000000` как цвет текста
- ❌ Холодные серые фоны
- ❌ Яркие / неоновые цвета
- ❌ Фиолетовый / синий как основной

## 7. ВИЗУАЛЬНЫЕ ПРАВИЛА

**Анимации:**
- `fade-in + translateY(24px → 0)` при появлении секций через `IntersectionObserver`
- `transition: opacity 0.7s ease, transform 0.7s ease`
- Hover кнопок: `transition: 0.3s`
- `prefers-reduced-motion` — при включённом отключать все анимации

**Сигнатурный элемент:**
Тонкая черта `1px solid var(--accent-main)` с `opacity: 0.35` между секциями — ритм страницы, не декор.

**Скругления:**
- Единый радиус `12px` на весь сайт — фото/медиа, кнопки, карточки
- Поля формы и разделители — без скругления
- Аватары, кнопки карусели, Play — `50%`

**Карточки услуг:**
- Фон `var(--surface-card)` = `#EFE2CF` (на тон светлее секции `#E8D9C5`), `border-radius: 12px`, `padding: 32px 28px`
- Акцент сверху: `border-top: 3px solid var(--accent-main)`
- Тянутся по высоте (`display: flex; flex-direction: column`), цена/кнопка прижаты вниз
- Никаких `box-shadow`

**Карусель (отзывы):**
- Листается нативным горизонтальным скроллом (`overflow-x: auto`, `scroll-snap-type: x mandatory`)
- Каждая карточка: `scroll-snap-align: start`
- Скроллбар скрыт через `::-webkit-scrollbar { display: none }`
- Кнопки ← → управляют скроллом через JS (`scrollBy`)
- На мобайле: свайп работает нативно, кнопки скрыть

**Поля формы:**
- Только `border-bottom: 1px solid var(--divider)`, без рамок со всех сторон
- Без `border-radius`, фон прозрачный
- Focus: `border-bottom-color: var(--accent-main)`
